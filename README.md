## Requirements
+ `Python` >= 3.9.7
+ `Pip` >= 21.2.3

## Installation
Clone our source code
```sh
$ git clone https://github.com/hiepnguyen3872/MapColoringProblem.git
$ cd MapColoringProblem
```

Install Dependencies
```sh
$ pip install -r requirements.txt
```

Run program

```sh
$ python main.py
```
